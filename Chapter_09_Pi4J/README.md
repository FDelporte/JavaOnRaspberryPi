# Chapter 9: Pi4J

This directory contains many examples that still use V1 of the Pi4J library. That's because these were created for the printed version of the book in 2020 and the early version of the ebook.

Many of these examples got replaced by newer implementations with Pi4J V4 and JBang. These are available on [github.com/Pi4J/pi4j-jbang](https://github.com/Pi4J/pi4j-jbang).
